## Summary

This patch release fixes two critical authoring issues: the Course Builder page rendering blank after navigating from Course Details (caused by an internal state reset clobbering the active course ID), and the assessment builder not appearing when "Edit Assessment" was clicked. It also blocks external live reviewers from accidentally downloading course resources. No new features; 130 new unit tests improve confidence in the test suite.

## üêõ Fixes

- **author / course-collection** ‚Äî blank Course Builder and assessment builder: `BehaviorSubject` seeded with `''` was clobbering `currentCourseId` on every subscriber, causing `save()` to call `/hierarchy/?mode=edit` (no ID) ‚Üí 404 ‚Üí blank page. Three guards added: skip empty emissions, recover course ID from URL if cleared, null-check before `readcontentV3`. (`243550b1`)
- **app-toc** ‚Äî block external live reviewers from downloading course resources (`a05cc3a0`)
- **my-content** ‚Äî DASHBOARD sidebar label now matches the page-title header (`d7a51f1d`)
